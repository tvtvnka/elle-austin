A little bit about me.

From Ohio to Oklahoma and back, my journey into tech started in the mid '90s when my mom brought home a PC. I loved researching and playing PC games for hours. I started learning how to code and create graphics when I joined Myspace and would change my layout and theme at least once a month and I eventually moved on to coding my own websites using free hosts.

That curiosity then opened a door that could never be shut. When I figured out how to build my first PC, I realized I found something that felt natural and would give me a great sense of purpose and accomplishment, and where I can use my creative skills and eye for detail to create unique and functional code.

Hand-Coded HTML

My focus is writing clean, well-formatted, semantic HTML5 by hand to make sure that the content is easy to read, easy to collaborate, trouble-shoot and accessible.

Well-Organized CSS

I pride myself on writing CSS that is easy to read and build on. I focus on keeping my CSS lean and fast to load, and I make it a habit to stay up to date on current best practices.
